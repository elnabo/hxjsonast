class Printer

package hxjsonast

Printer for the Json values. Outputs a valid JSON string. Positions are ignored.

Static methods

staticprint (json:Json, ?space:String):String

Output a JSON string for given json value.

If space is specified, the output will be pretty-printed using given space value for indentation.
